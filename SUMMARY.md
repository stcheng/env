# Repo Summary

Small config-and-utilities repo with terminal and window manager settings plus a single helper script.

## Contents

- `RectangleConfig.json`: Rectangle (macOS window manager) configuration with shortcut bindings and layout preferences.
- `iterm.json`: iTerm2 profile configuration (colors, fonts, keyboard map, logging, etc.).
- `ShuotianiTermConfig.json`: Another iTerm2 profile configuration, similar structure to `iterm.json`.
- `misc/calculate_mean.py`: Python 2 script that reads a CSV of `key,value` pairs and prints the mean per unique key.
- `LICENSE`: Project license.

## Notes

- Global git aliases were configured in `~/.gitconfig` (outside this repo).
